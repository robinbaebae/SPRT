## Slide 2: 문제 정의

### "Rate Limit 걸려봐야 알게 되지 않나요?"

| 😵‍💫 | Claude Code를 쓰다가 갑자기 **Rate Limit** — 작업 흐름이 끊긴다 |
|---|---|
| ❓ | 지금 세션 사용량이 **얼마나 남았는지** 확인할 방법이 없다 |
| 📊 | 주간 한도도 있는데, **어디서 확인하지?** |
| ⏰ | 리셋까지 **얼마나 기다려야 하는지** 모른다 |

> Claude Code 사용자라면 누구나 겪는 문제입니다.

## Slide 3: 솔루션

### SPRT = 메뉴바 사용량 모니터

**코딩하면서 메뉴바 한 번 보면 끝.**
Rate Limit, 주간 한도, 리셋 타이머를 실시간으로 확인합니다.

| 기능 | 설명 |
|------|------|
| **메뉴바 트레이** | 현재 5h 세션 사용률을 **%** 로 실시간 표시 |
| **팝오버** | 트레이 클릭 → 현재 세션 요약 즉시 확인 |
| **대시보드** | 더블클릭 → Rate Limits (5h/7d) + 주간 활동 차트 |
| **알림** | 80% 경고, 100% 도달 시 macOS 알림 |

## Slide 4: AI 활용 핵심 — 데이터 수집 아키텍처

### 공식 API + 로컬 파싱 하이브리드

```
┌─ 데이터 소스 1: OAuth API ──────────────┐
│  Claude Code 인증 정보 재사용             │
│  → Anthropic API Rate Limit Headers     │
│  → 5h 세션 한도, 리셋 시간               │
└──────────────────────────────────────────┘

┌─ 데이터 소스 2: 로컬 파일 ───────────────┐
│  ~/.claude/projects/ JSONL 파싱          │
│  → 일별 메시지 수, 토큰 사용량            │
│  → 세션 활동 기록                        │
│  ※ 데이터가 외부로 전송되지 않음           │
└──────────────────────────────────────────┘

┌─ 실시간 갱신 ────────────────────────────┐
│  Rust 파일 워처가 ~/.claude/ 변경 감지    │
│  → 자동 새로고침 (폴링 없음)              │
└──────────────────────────────────────────┘
```

**핵심: 별도 API 키 불필요, Claude Code 인증 정보만으로 동작**

## Slide 5: 주요 기능

### 메뉴바에서 시작하는 사용량 관리

| 기능 | 설명 |
|------|------|
| **실시간 속도 제한** | 현재 사용률 + 리셋 카운트다운, 제한 임박 시 알림 |
| **주간 사용량** | All Models / Sonnet 주간 한도 모니터링 |
| **7일 활동 차트** | 일별 메시지 수 시각화, 사용 패턴 파악 |
| **메뉴바 상주** | 클릭 → 팝오버, 더블클릭 → 대시보드, 우클릭 → 메뉴 |
| **라이트/다크 모드** | 시스템 설정에 따라 자동 전환 |
| **macOS 알림** | 80% 경고 + 100% Rate Limit 도달 알림 |

## Slide 6: 기술 스택

### Rust + React — 가볍고 빠르게

```
┌──────── Frontend ────────┐  ┌────── Backend (Rust) ──┐
```

## Slide 7: 기존 방식 vs SPRT

| 카테고리 | 기존 방식 | SPRT |
|---------|:--------:|:----:|

| 앱 크기 | — | **5.3MB (초경량)** |
| 데이터 프라이버시 | — | **100% 로컬 처리** |

## Slide 8: 설치 & 사용

### 2분이면 시작

1. **Claude Code CLI** 설치 및 로그인
   ```bash
   npm install -g @anthropic-ai/claude-code
   claude  # 로그인
   ```

2. **SPRT DMG 다운로드** (5.3MB)
   → Applications 폴더에 드래그

3. **보안 해제** (최초 1회)
   ```bash
   xattr -cr /Applications/SPRT.app
   ```

4. **완료!** 메뉴바에 사용량이 표시됩니다.
