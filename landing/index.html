How It Works

Powered by official APIs

OAuth Rate Limit Headers

SPRT uses your local Claude Code OAuth credentials to fetch real-time usage data from Anthropic's API. The same unified rate limit headers that Claude Code itself uses.

Local JSONL Parsing

Activity data comes from parsing your local ~/.claude/projects/ JSONL files. Message counts, token usage, and session activity are all computed locally. Your data never leaves your machine.

Getting Started

Up and running in 2 minutes

macOS 13+ Apple Silicon (M1/M2/M3/M4) Claude Code CLI Claude Pro / Max plan
1

Install Claude Code CLI

SPRT reads your Claude Code credentials to fetch usage data. Install and log in first:

npm install -g @anthropic-ai/claude-code

Then run claude and follow the login prompts to authenticate with your Anthropic account.

2

Download & Install SPRT

Download DMG (5.3MB)

Open the downloaded DMG file and drag SPRT into your Applications folder.

3

Bypass macOS Security Warning

SPRT is not yet signed with an Apple Developer certificate, so macOS will show a security warning on first launch. This is normal for open-source apps distributed outside the App Store.

Option A — Terminal (recommended):

Open Terminal and run:

xattr -cr /Applications/SPRT.app

Then double-click SPRT to open normally.

Option B — System Settings:

1. Try opening SPRT (it will be blocked)
2. Go to System Settings → Privacy & Security
3. Scroll down and click "Open Anyway" next to the SPRT message
4. Click "Open" in the confirmation dialog

4

You're All Set!

SPRT will appear in your menu bar with a small logo and your current usage percentage.

Single click the tray icon → Quick popover with current session
Double click → Full dashboard with weekly limits & activity chart
Right click → Open Dashboard or Quit

SPRT runs as a menu bar app — there's no Dock icon. To quit, right-click the tray icon and select "Quit".
