How It Works

공식 API 기반으로 동작

OAuth Rate Limit Headers

로컬에 저장된 Claude Code OAuth 인증 정보를 사용하여 Anthropic API에서 실시간 사용량 데이터를 가져옵니다. Claude Code가 사용하는 것과 동일한 통합 속도 제한 헤더를 활용합니다.

로컬 JSONL 파싱

활동 데이터는 로컬 ~/.claude/projects/ JSONL 파일을 파싱하여 가져옵니다. 메시지 수, 토큰 사용량, 세션 활동이 모두 로컬에서 계산됩니다. 데이터가 외부로 전송되지 않습니다.

Getting Started

2분이면 시작할 수 있습니다

macOS 13+ Apple Silicon (M1/M2/M3/M4) Claude Code CLI Claude Pro / Max 플랜
1

Claude Code CLI 설치 및 로그인

SPRT는 Claude Code의 인증 정보를 사용해 사용량 데이터를 가져옵니다. 먼저 CLI를 설치하고 로그인하세요:

npm install -g @anthropic-ai/claude-code

설치 후 claude를 실행하고 로그인 안내에 따라 Anthropic 계정으로 인증하세요.

2

SPRT 다운로드 및 설치

DMG 바로 다운로드 (5.3MB)

다운로드한 DMG 파일을 열고 SPRT를 Applications 폴더로 드래그합니다.

3

macOS 보안 경고 해제하기

SPRT는 아직 Apple 개발자 인증서로 서명되지 않아서, 처음 실행할 때 macOS가 보안 경고를 표시합니다. App Store 외부에서 배포되는 오픈소스 앱에서는 정상적인 현상입니다.

방법 A — 터미널 (추천):

터미널을 열고 아래 명령어를 실행하세요:

xattr -cr /Applications/SPRT.app

이후 SPRT를 정상적으로 실행할 수 있습니다.

방법 B — 시스템 설정:

1. SPRT를 실행 시도합니다 (차단됨)
2. 시스템 설정 → 개인정보 보호 및 보안으로 이동
3. 하단에 SPRT 관련 메시지 옆 "그래도 열기" 클릭
4. 확인 팝업에서 "열기" 클릭

4

설정 완료!

메뉴바에 SPRT 로고와 현재 사용량 퍼센테이지가 표시됩니다.

클릭 → 현재 세션 요약 팝오버
더블클릭 → 전체 대시보드 (주간 한도 & 활동 차트)
우클릭 → 대시보드 열기 또는 종료

SPRT는 메뉴바 전용 앱입니다 — Dock 아이콘이 없습니다. 종료하려면 트레이 아이콘을 우클릭하고 "Quit"을 선택하세요.

더 이상 사용량을 추측하지 마세요.

Claude Code 사용량을 정확히 파악하세요.
